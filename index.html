Kako koristiti Tarello

  • Bočna traka — gumb ☰ (gore lijevo) otvara traku sa sekcijama za brze bilješke (Važni kontakti, Bitni ugovori...). "+ Dodaj" dodaje stavku, klik na stavku je uređuje, "+ Nova sekcija" pravi novu sekciju, klik na naziv sekcije je preimenuje.
  • Ploče — pregled svih tvojih ploča. Klik otvara ploču, "+ Nova ploča" pravi novu. Ikonom 📷 na pločici postavi sliku kao pozadinu ploče (ponovni klik na 📷 je uklanja).
  • Linkovi na ploči — u otvorenoj ploči, pored naziva, gumb + dodaje link (Facebook, Instagram, mail, telefon, bilo koja stranica). Aplikacija sama prepozna poznate servise i stavi ikonu; za ostale biraš emoji. Klik na ikonu otvara link, ✕ na ikoni je uklanja.
  • Kartice — klikni "+ Dodaj karticu": upišeš naziv i pritisneš Enter — polje ostaje otvoreno za sljedeći unos (Esc zatvara). Kružić na kartici jednim klikom označava gotovo/otvoreno. Klik na karticu otvara uređivanje (labela, članovi, rok).
  • Drag & drop — povuci karticu mišem u drugu listu.
  • Moji zadaci — prikazuje sve kartice sa svih ploča. Prekidačem "Samo moji" filtriraj na kartice s tvojim inicijalima (postavi ih u Adminu).
  • Kalendar — kartice kojima si dodao rok prikazuju se na datumu roka (teal boja).
  • Podsjetnici — klikni na bilo koji dan u kalendaru da dodaš podsjetnik 🔔 (narančasto). Klik na postojeći podsjetnik ga uređuje ili briše.
  • Spremanje — sve se automatski sprema u ovaj browser (localStorage). Za sigurnosnu kopiju koristi Export u Adminu.
  • Automatski backup — u Adminu klikni "Poveži backup fajl" (Chrome/Edge): odabereš fajl na disku jednom, i svaka promjena se od tada automatski zapisuje u njega.
  • QR kod — generiraj QR za javni link aplikacije i skeniraj mobitelom da je otvoriš na telefonu.
  • Vraćanje obrisanog — nakon brisanja se dolje na 8 sekundi pojavi gumb "↩ Vrati" za brzo vraćanje. Ako ga propustiš — sve obrisano čeka u Admin → Smeće, odakle ga možeš vratiti bilo kad (čuva se zadnjih 50 stavki).
  • Export u Excel/PDF — u Adminu izvezi sve taskove u Excel tablicu ili PDF dokument (PDF: u dijalogu ispisa odaberi "Spremi kao PDF").

⇅ GitHub sinkronizacija

Gumb "Sinkaj" šalje podatke direktno na GitHub jednim klikom. Za to treba GitHub token (upute ispod) — sprema se samo na ovom uređaju, nikad u javni kod.

Kako napraviti token (jednom): GitHub → klik na svoju sliku → Settings → skroz dolje Developer settings → Personal access tokens → Fine-grained tokens → Generate new token → Repository access: Only select repositories → Aplikacija → Permissions → Contents: Read and write → Generate → kopiraj token (počinje s github_pat_) i zalijepi gore.
